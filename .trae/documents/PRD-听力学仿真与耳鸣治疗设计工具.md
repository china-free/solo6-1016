## 1. 产品概述

在线听力学仿真与耳鸣声治疗设计工具，面向听力学专业人员和耳鸣患者，提供基于心理声学模型的个性化掩蔽声治疗方案生成与试听功能。

- 核心目的：将听力检测数据转化为可操作的治疗参数，通过科学算法设计个性化耳鸣掩蔽声
- 目标用户：耳鼻喉科医生、听力师、耳鸣患者、康复治疗师
- 市场价值：填补耳鸣声治疗数字化工具空白，使专业治疗方案可定制、可试听、可微调

## 2. 核心功能

### 2.1 功能模块

1. **听力图输入模块**：标准频率点听阈输入、听觉动态范围计算、听力图可视化
2. **耳鸣特征描述模块**：耳鸣频率、响度、类型（纯音/窄带噪声/宽带噪声）输入
3. **掩蔽声参数计算模块**：基于心理声学模型的最佳掩蔽参数自动生成
4. **音频合成与试听模块**：Web Audio API实时合成白噪声、粉红噪声、陷波音乐等
5. **参数微调模块**：频率中心、带宽、声级的实时调整与即时反馈

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主工作台 | 听力图输入区 | 8个标准频率点(125Hz-8kHz)的左右耳听阈输入，支持滑块与数字输入 |
| 主工作台 | 动态范围显示区 | 实时计算并显示各频率点听觉动态范围(阈值→不适阈) |
| 主工作台 | 听力图可视化 | Canvas绘制标准听力图曲线，双耳区分显示 |
| 主工作台 | 耳鸣特征配置 | 频率选择、响度评分(1-10)、耳鸣类型下拉选择 |
| 主工作台 | 掩蔽参数结果 | 显示计算后的中心频率、带宽、声级参数 |
| 主工作台 | 音频合成控制 | 噪声类型选择、播放/暂停、音量控制 |
| 主工作台 | 参数微调面板 | 频率±500Hz微调、带宽±1000Hz微调、声级±20dB微调 |

## 3. 核心流程

用户输入听力图数据 → 系统计算听觉动态范围 → 用户描述耳鸣特征 → 后端调用心理声学掩蔽模型 → 返回最佳掩蔽声参数 → 前端通过Web Audio API合成音频 → 用户试听 → 用户微调参数 → 实时更新音频

```mermaid
    D --> E["生成掩蔽参数"]
    E --> F["Web Audio合成音频"]
    F --> G["用户试听"]
    G --> H["微调参数"]
    H --> F
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深海蓝(#1e3a5f) - 传达医疗专业感与科技感
- **辅助色**：薄荷绿(#2dd4bf) - 强调治疗与舒缓，珊瑚橙(#fb923c) - 警示与重要操作
- **中性色**：冷灰系列(slate) - 保证内容可读性
- **按钮风格**：圆角8px，渐变背景，悬停微上浮阴影效果
- **字体**：思源黑体(Noto Sans SC) - 中文显示清晰，搭配专业等宽字体展示参数数值
- **布局风格**：三栏卡片式布局 - 左侧听力输入、中间耳鸣配置与结果、右侧音频控制
- **图标风格**：Lucide图标库，线条风格，统一2px描边

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 主工作台 | 听力图输入区 | 双耳双列布局，频率标签+滑块+dBL数值输入框，动态范围进度条 |
| 主工作台 | 听力图可视化 | Canvas坐标系，横轴频率对数刻度，纵轴听阈dB HL，红蓝色区分双耳 |
| 主工作台 | 耳鸣特征配置 | 卡片容器，标签化输入组，类型选择使用图标按钮组 |
| 主工作台 | 掩蔽参数结果 | 大号数值展示卡，参数标签+数值+单位，渐变边框高亮 |
| 主工作台 | 音频合成控制 | 大型播放按钮居中，波形可视化条，噪声类型切换Tab |
| 主工作台 | 参数微调面板 | 三滑块横向排列，±按钮快捷调整，实时数值显示 |

### 4.3 响应性
- 桌面优先设计(1280px+)：三栏并排布局
- 平板(768-1279px)：上下两栏，听力图与音频控制纵向排列
- 移动(<768px)：单栏滚动，听力图缩放适配，触控优化滑块
